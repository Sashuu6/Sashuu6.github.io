Setup Wake on LAN on Local Server

Wake On LAN
  • 04 Oct 2018

Why Wake on LAN? Wake on LAN is program to wake a system which is under deep sleep. That is, the system is shut down with power access. A system with an enabled Wake-on-LAN feature can be turned on using a special packet. This special packet is sent by the client remotely.

To implement Wake-On-LAN on the server, we need to install ethtool on our server and wakeonlan commanline program on our personal system.

First we will setup our server:-

  1. Enable wake-on-LAN on BIOS settings. Enter into BIOS and search for wake on LAN and enable it.
  2. Make sure the ethernet cable connect from the server to the switch or router has all the 8 pins connected. This is because the magic packet is sent via a dedicatd pin, usually last 2 pins.
  3. Install ethtool into the server.
  4. (Ubuntu / Debian)
  5. sudo apt install ethtool
  6. (Fedora)
  7. sudo dnf install ethtool
  8. (RHEL / CentOS)
  9. sudo yum install ethtool
  10. Now let's find the name of ethernet port present in the system. This command will also reveal the mac address.
  11. ip a
  12. The above command will output the ethernet port name (Lets consider the ethernet name to be enp8s0).
  13. Let's check if ethtool detects the port.
  14. sudo ethtool enp8s0
  15. Now, let's activate wake on lan.
  16. sudo ethtool -s enp8s0 wol g
  17. Now, we will setup wake on client on your system. i.e, the system that will be used to wake the server.
  18. (Ubuntu / Debian)
  19. sudo apt install etherwake
  20. (Mac)
  21. sudo brew install wakeonlan
  22. (Fedora / RHEL / CentOS)
  23. wget http://gsd.di.uminho.pt/jpo/software/wakeonlan/downloads/wakeonlan-0.41-0.fdr.1.noarch.rpm 
    rpm -Uvh wakeonlan-0.41-0.fdr.1.noarch.rpm

  24. Now, we can wake the system using the wakeonlan command.

  25. Execute the following command.

  26. wakeonlan mac-address
